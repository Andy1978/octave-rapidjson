# octave-rapidjson

[GNU Octave](https://www.gnu.org/software/octave/) wrapper around [RapidJSON](http://rapidjson.org/)

## Ready for test

* `load_json (json_string)`. If you want to parse from file use `load_json (fileread (filename))`
